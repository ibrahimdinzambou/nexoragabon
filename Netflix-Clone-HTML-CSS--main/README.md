# Netflix Clone (HTML & CSS)

Ce projet est un clone simple de la page d'accueil de Netflix, créé uniquement avec HTML et CSS. Il reproduit la mise en page de base, la barre de navigation et les sections mises en avant du site Netflix. Le projet met l'accent sur la pratique du développement web front-end, avec un focus sur le design responsive.

## Fonctionnalités
- Reproduit le design de la page d'accueil Netflix
- Mise en page réactive qui s'adapte à différentes tailles d'écran
- Construit en HTML et CSS purs (sans JavaScript ni backend)

## Technologies utilisées
- **HTML5**
- **CSS3**
- **Visual Studio Code** pour le développement
- **Chrome** pour les tests
- **Extension Live Server** pour le rechargement en direct pendant le développement

## Installation
1. Clonez ce dépôt sur votre machine locale :
    ```bash
    git clone https://github.com/your-username/netflix-clone-html-css.git
    ```
2. Ouvrez le projet dans Visual Studio Code :
    ```bash
    code netflix-clone-html-css
    ```
3. Installez l'extension **Live Server** dans Visual Studio Code, si ce n'est pas déjà fait.

## Utilisation
1. Faites un clic droit sur le fichier `index.html` dans Visual Studio Code et sélectionnez **Open with Live Server**.
2. Le projet s'ouvrira dans votre navigateur par défaut (de préférence Chrome) et se mettra à jour automatiquement lorsque vous modifierez le code.
